# Smart Waste Sorting and Pickup Request System

A desktop application built with Java and JavaFX that
allows apartment caretakers to submit waste pickup
requests and city waste management admins to schedule
and track pickups.

## SDG Alignment

- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production

## Tech Stack

- Java JDK 21
- JavaFX 21
- MySQL 8.0
- JDBC

## Project Structure

WastePickup/

├── model/ → Java model classes

├── dao/ → Data access objects

├── ui/ → JavaFX screens

│ ├── views/ → FXML layouts

│ └── controllers/ → Screen controllers

├── util/ → Database connection

└── lib/ → External libraries
